## 🎯 Описание проекта

Проект представляет собой аналитическое исследование данных о розничных продажах. Цель — выявить ключевые товарные категории, оценить эффективность продаж и провести многофакторный анализ.

## 🚀 Установка и запуск

### Требования
- Python 3.8 или выше

### Установка зависимостей

```bash
pip install pandas numpy matplotlib seaborn openpyxl
```
### Запуск скриптов

```bash
# Анализ самой ходовой товарной группы
python MostPopularProductGroup.py

# Анализ распределения по подкатегориям
python SalesSubcategories.py

# Расчёт среднего чека
python AverageBill.py

# Анализ промо в категории "Сыры"
python Promo.py

# Расчёт маржи по категориям
python MarginByCategory.py

# ABC-анализ
python ABC_analys.py
```

## 📈 Задачи и результаты

### Задачи
### 1. Анализ продаж

```bash
- Определена товарная категория с наибольшим объёмом продаж (в штуках)
- Построена сводная таблица продаж по категориям
- Визуализировано распределение с помощью bar chart
- Проанализировано распределение продаж внутри категорий (level1 → level2)
```

### 2. Анализ ключевых метрик
- Рассчитан средний чек на заданную дату
- Выявлена доля промо-продаж в категории Сыры:
  - расчёт доли в штуках
  - визуализация через pie chart

### 3. Анализ маржинальности
- Рассчитана маржа по категориям (level1):
  - в абсолютном выражении (рубли)
  - в относительном (%)
- Построены горизонтальные bar chart для наглядного сравнения категорий

### 4. ABC-анализ ассортимента
- Проведён ABC-анализ:
  - по количеству продаж
  - по выручке
- Сформирована комбинированная сегментация (например: A–B, A–C и т.д.)
- Выполнена оценка вклада категорий в общий результат
